一家样式雷，半部古建史

以官式建筑色系与样式雷营造文脉为底，构建庄重、典雅、专业、清晰的教务工作系统。下含设计令牌、品牌组件、五个关键页面高保真稿与组件库一览——浅色为主，附深色模式令牌。

全省双高 · 古建专业群 100 名教师自助录入 服务端渲染 · 轻量主题

设计前提与判断（请确认）

气质定位：博物馆数字平台式的克制高级感——朱红+鎏金作"点"，宣纸米作"面"，纹样仅用于分隔/徽章/底纹，绝不堆砌。
取名占位：示意稿暂用"人居环境与艺术学院 / 教学资料管理系统"，请替换为真实校名与系统全称。
影像策略：角楼/烫样大图用占位框标注，请提供授权实景照（横构图、暖调）；我不手绘古建以免失真。
技术落地：全部用 CSS 变量(设计令牌)主题化，少特效、可服务端渲染，旧机可跑；图表用纯 CSS 绘制。
无障碍：正文对比度≥4.5:1，关键操作有清晰金色焦点态，导航与表单全键盘可达。
响应式：录入表单分步、字段带示例与校验，小屏单列回流，触控目标≥44px。
